# Changelog

Все заметные изменения в проекте SKOLKOVO Coaching Materials Processor.

## [0.2.1] - 2025-11-26

### Исправлено
- **GUI PDF Processing**: Исправлена критическая ошибка обработки PDF через GUI
  - Убран некорректный параметр `page_range` из вызова `pdf_to_images()`
  - Заменены несуществующие методы `_create_batches()`, `_process_batch()` на корректный `process_pages()`
  - Заменён несуществующий метод `_create_markdown_report()` на `_save_as_markdown()`
  - Исправлен ключ результата: `source_file` → `source_pdf`
  - Добавлена корректная обработка путей файлов для Gradio 5.x (строки вместо объектов)

### Изменено
- Логика фильтрации страниц: теперь применяется ПОСЛЕ конвертации PDF в изображения
- Упрощена структура кода обработки PDF в gui.py

## [0.2.0] - 2025-11-25

### Добавлено
- GUI интерфейс на базе Gradio 5.x
- Обработка PDF через Claude Vision API
- Транскрибация аудио через OpenAI Whisper API
- Поддержка различных режимов анализа PDF (summary, key_points, full_analysis)
- Автоматическое сохранение результатов в Markdown

### Технические детали
- `PDFVisionProcessor` класс для работы с PDF
- `AudioTranscriber` класс для транскрибации аудио
- Батчевая обработка страниц PDF для оптимизации API-вызовов

## [0.1.0] - 2025-11-24

### Добавлено
- Начальная структура проекта
- Базовая обработка PDF файлов
- Конфигурация через .env файлы
